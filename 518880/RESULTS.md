# 518880 主要发现

## Spread 2tick 的状态特征

- `spread=2 tick` 是一个相对复杂的状态，不适合简单视为 `spread=1 tick` 的放大版。
- 相较于 `spread=1 tick`，`spread=2 tick` 状态下盘口不平衡率更低。
- 仅用不平衡率去预测 `spread=2 tick` 时刻的价格方向，效果弱于 `spread=1 tick`。
- 但 `spread=2 tick` 往往对应未来更强的价格变动，说明它对“后续波动强度”更敏感，对“即时方向”反而没那么直接。
- 2tick状态几乎不能保留，会立即回到1tick。

## 1tick 扩到 2tick 的过渡现象

- 当 `spread` 从 `1 tick` 扩大到 `2 tick` 前，大约前 `2s` 的不平衡率会明显抬升。真正进入 `2 tick` 后，不平衡率又会回到相对正常的水平。
- 在扩大前，L1 深度会不断减小，扩大后 `1s`恢复正常水平。
- 在到达 `2 tick` 的那个时刻前 `2s`，主动成交量会出现明显跳升; 扩大后的交易量水平会略低于交易前水平。

## 扩大原因和结果

- 扩大主要是买盘卖盘最优价格中的一个被消耗，也有极少数同时耗尽的。结合主动成交量，应该是主动成交而非撤单。


## 当前解释

- `spread` 扩大更像是主动成交把最优价位吃空，而不只是被动挂单撤退。
- 因此，`1tick -> 2tick` 更应被理解为一次短时流动性被消耗的事件，而不是单纯的静态宽价差状态。

## 扩价前 2s 状态对未来的影响

- 新增脚本：`518880/code/spread2_pre2s_impact.py`
- 样本定义：只看 `spread` 从 `1 tick -> 2 tick` 的事件，并要求扩价前连续 `2s` 都处于 `spread=1 tick`。
- 样本区间：`2025-12-01` 到 `2026-04-25`，共 `49` 个交易日，`30830` 个扩价事件，其中 `25700` 个满足“前 2s 干净”的筛选条件。

### 影响本身

- 扩价事件后，下一次非零 mid move 向上的概率约 `51.1%`，中位等待时间 `1s`，说明扩价后很快就会继续发生价格动作，但即时方向优势很弱。
- 未来 `5s / 10s / 30s` 的平均绝对 mid 变动分别约为 `1.20 / 1.70 / 3.03 tick`，说明扩价更稳定地对应“后续波动增强”，而不是强方向性。
- 从均值看，未来 `5s / 10s / 30s` 的平均 mid 收益分别约为 `-0.009 / -0.013 / -0.043 tick`，整体略偏负，但量级很小。

### 前 2s 能否预测方向

- 能，但比较弱，且主要来自盘口不平衡，而不是成交量。
- 用扩价前 `2s` 与 `1s` 的盘口/成交联合特征预测：
  - `next_up` 的测试 AUC 约 `0.552`
  - `5s` 方向测试 AUC 约 `0.564`
  - `10s` 方向测试 AUC 约 `0.560`
  - `30s` 方向测试 AUC 约 `0.543`
- 方向信息最强的单因子不是主动成交，而是扩价前 `1s` 的 `L3/L5 imbalance`：
  - `pre1_l3_imbalance` 对 `5s` 方向的 AUC 约 `0.550`
  - `pre1_l3_imbalance` 对 `10s` 方向的 AUC 约 `0.546`
  - `pre1_l3_imbalance` 对 `30s` 方向的 AUC 约 `0.533`
- `pre1_l1_imbalance` 也有作用，但明显弱于 `L3`，说明扩价前的方向信息不只在最优一档，更像是浅层盘口整体偏斜。

### 一个更直接的读法

- 如果扩价前 `1s` 买盘更强（`pre1_l1_imbalance > 0`），未来 `5s / 10s / 30s` 向上概率约为 `50.2% / 50.6% / 50.9%`。
- 如果扩价前 `1s` 卖盘更强（`pre1_l1_imbalance < 0`），未来 `5s / 10s / 30s` 向上概率约为 `46.8% / 47.8% / 49.2%`。
- 两边差值在 `5s~10s` 约 `2~4` 个百分点，说明它更适合作为“微弱方向过滤器”，不适合作为单独强信号。
